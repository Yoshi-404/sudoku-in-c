# Sudoku in C 🧩

[![Language: C](https://img.shields.io/badge/Language-C-blue.svg)](https://en.wikipedia.org/wiki/C_(programming_language))

*(For the Portuguese version, scroll down / Para a versão em português, role a página para baixo)*

A terminal-based Sudoku game written in C. This project was developed to practice fundamental programming logic and data persistence. 

The game allows you to play, validate row/column/quadrant inputs, automatically solve steps, and save or load your progress using file manipulation (binary and `.txt` files).

## How to compile and play

To run this project locally, you will need a C compiler installed on your machine, such as GCC.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/](https://github.com/)Yoshi-404/sudoku-in-c.git
   ```


3. **Navigate to the project directory:**
  ```bash
  cd sudoku-in-c
  ```

3. **Compile the code:**
  ```bash
  gcc sudoku.c -o sudoku
  ```


4. **Run the game:**
* On Linux/macOS: `./sudoku`
* On Windows: `sudoku.exe`

# Sudoku em C 🧩

Um jogo de Sudoku jogável via terminal escrito em C. Este projeto foi desenvolvido para praticar lógica de programação e persistência de dados.

O jogo permite jogar, validar entradas (linhas, colunas e quadrantes), resolver passos automaticamente, além de salvar e carregar o progresso do usuário utilizando manipulação de arquivos (arquivos binários e de texto `.txt`).

## 🚀 Como compilar e jogar

Para rodar este projeto localmente, você precisará de um compilador C instalado na sua máquina, como o GCC.

1. **Clone o repositório:**
```bash
git clone [https://github.com/](https://github.com/)Yoshi-404/sudoku-in-c.git
```


2. **Acesse a pasta do projeto:**
```bash
cd sudoku-in-c
```


3. **Compile o código:**
```bash
gcc sudoku.c -o sudoku
```


4. **Execute o jogo:**
* No Linux/macOS: `./sudoku`
* No Windows: `sudoku.exe`
